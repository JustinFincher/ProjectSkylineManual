## Location

Two ways to set a location.

- Pan to move.
- Make a search in "TAP TO SEARCH" dialog.

## Zoom

Pinch to zoom.

## Rotation

Rotate the screen with two fingers.

## Color Filter

You can disable the vibrant color filter in the settings panel.
